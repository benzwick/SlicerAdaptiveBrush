SlicerAdaptiveBrush Documentation
==================================

SlicerAdaptiveBrush is a 3D Slicer extension providing an adaptive brush segment editor effect.
The brush automatically segments regions based on image intensity similarity, adapting to image
features (edges, boundaries) rather than using a fixed geometric shape.
